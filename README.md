# Sanity Blog Project

A modern blogging platform built with **React**, **Sanity CMS**, and **Tailwind CSS**.  
This project allows users to create, manage, and display dynamic blog posts with categories, responsive layouts, and a clean UI.

---

## 🚀 Features

- ✍️ **Content Management** with Sanity Studio
- 📖 **Dynamic Blog Posts** with titles, images, and categories
- 📱 **Responsive Design** powered by Tailwind CSS
- ⚡ **Fast and Optimized** with React + Vite
- 🔗 Easy to extend with more features like comments, tags, or search

---

## 🛠️ Tech Stack

- **React.js** – frontend framework
- **Sanity.io** – headless CMS for content
- **Tailwind CSS** – styling
- **React Router** – navigation
- **Vite** – fast development build tool
